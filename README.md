# flask-jsonapi

JSONAPI 1.0 server implementation for Flask.

## Running tests

```
virtualenv -p python3.6 ~/flask-jsonapi-virtualenv
. ~/flask-jsonapi-virtualenv/bin/activate
pip install -r base_requirements.txt
pip install -U pytest==3.0.5
pytest
```
